# New Features

## Priority Guidelines

When adding new features, follow this priority order for choosing libraries or implementation methods:

| Priority | Option                     | Reason               |
| -------- | -------------------------- | -------------------- |
| 1        | Electron native APIs       | Native + performant  |
| 2        | Existing project libraries | Consistent + simple  |
| 3        | Popular external libraries | Reliable + typed     |
| 4        | Custom implementation      | Flexible last resort |
